# IncomeFlowlance Backend

REST API backend for the IncomeFlowlance freelance income dashboard.

## 🎯 Purpose

Handles:

- Stripe OAuth authentication
- JWT token management
- Revenue data aggregation
- Client information retrieval
- Payment status tracking
- Data export functionality

## 🛠️ Tech Stack

- **Framework:** Spring Boot 3.x
- **Language:** Java 17+
- **Database:** PostgreSQL
- **Authentication:** JWT + Spring Security
- **External APIs:** Stripe API v1
- **Build Tool:** Maven

## 📋 Prerequisites

- Java 17 or higher
- Maven 3.8+
- PostgreSQL 12+
- Stripe Account (test/live keys)
